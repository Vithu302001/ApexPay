# ApexPay

ApexPay is a Java backend project that I am building to strengthen my backend engineering skills. The project is being developed incrementally, starting with Core Java and later evolving into a Spring Boot application with database integration, caching, asynchronous processing, and REST APIs.

## Features Implemented

- Account management
- Deposit transactions
- Withdrawal transactions
- Transfer transactions
- Account repository using HashMap
- Object-oriented design using inheritance and abstraction

## Tech Stack

- Java
- Maven
- Git
