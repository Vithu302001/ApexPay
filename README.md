# ApexPay

ApexPay is a backend application built to strengthen my Java backend engineering skills through a project-based learning approach. The project starts with Core Java and is progressively enhanced into a production-style Spring Boot application featuring database integration, REST APIs, caching, asynchronous processing, and other backend technologies.

## Current Features

- Account creation and management
- Deposit transactions
- Withdrawal transactions
- Money transfers between accounts
- Transaction history
- Repository-Service layered architecture
- Polymorphic transaction processing
- In-memory data storage using `HashMap` and `ArrayList`

## Architecture

```
Main
   │
   ▼
BankService
   ├── AccountRepository
   └── TransactionRepository
           │
           ▼
        Domain Models
```
